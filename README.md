🚀 16-Week Cloud Engineering Bootcamp

Overview

Welcome to my 16-Week Cloud Engineering Bootcamp.

This repository documents my journey to becoming a Senior Cloud Engineer / Senior Backend Engineer by strengthening my expertise in cloud architecture, backend development, DevOps, system design, and interview preparation.

The goal is not only to earn cloud certifications but also to build production-quality applications that demonstrate real-world engineering skills.

⸻

🎯 Bootcamp Objectives

By the end of this 16-week program, I will:

* Pass the AWS Certified Solutions Architect – Associate (SAA-C03) certification.
* Build three production-quality cloud projects.
* Master AWS architecture and cloud best practices.
* Strengthen Java 21 and Spring Boot expertise.
* Improve system design and software architecture skills.
* Gain hands-on experience with Docker, Kubernetes, and Terraform.
* Build CI/CD pipelines using GitHub Actions.
* Prepare for Senior Software Engineer and Senior Cloud Engineer interviews.
* Develop a professional GitHub portfolio that showcases enterprise-level projects.
